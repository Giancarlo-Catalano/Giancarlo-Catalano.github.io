Instructions for Teachers
RSS feeds act like a subscription to a website. They allow you to pull the latest announcements, articles, or news from external websites directly into your Canvas modules.

How to use this tool:
1. Find the RSS Link (URI) from your chosen website (usually ends in .xml or .rss).
2. Paste it below and configure your layout.
3. Copy the generated Embed Code.
4. In Canvas LMS, edit your page, switch to the "HTML Editor" (usually an icon with ... at the bottom right), and paste the code. Save the page.
Advanced: Load Previous Version

Need to edit a feed you already made? Paste the iframe embed code or link here to reload your settings.
